Diseño desde el Código

Este sitio se desarrolla para aplicar los contenidos aprendidos en clases del curso Diseño desde el Código del año 2022. El curso comprende los conceptos y recursos de código HTML Y CSS básicos que permitan el diseño y escritura de un sitios web. La entrega final del curso contempla esta sitio web personal, como también un sitio grupal sobre alguna problemática medioambiental.

Contenido aprendido en el curso

  1. Front-end
  2. HTML
  3. CSS
  4. Uso de contenedores
  5. Uso de GitHub y Visual Studio Code
  6. Uso de grillas para la diagramación de un sitio
  7. Recursos como bootstrap, repositorios, Google fonts
